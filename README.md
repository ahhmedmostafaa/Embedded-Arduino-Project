# Embedded System Project (Arduino)

An embedded systems project built on Arduino, demonstrating hardware-software integration.

## Overview
This project implements an embedded control system using Arduino, developed as part of an Embedded Systems course.

## Tech Stack
- Arduino (C/C++)
- Embedded hardware components

## Files
- `final_project/final_project.ino` — main Arduino sketch
- `final project emb.pdf` / `.docx` — full project documentation (circuit design, components, logic)

## How to Run
1. Open `final_project.ino` in the Arduino IDE
2. Connect the required hardware components (see documentation for the circuit diagram)
3. Upload the sketch to your Arduino board
